Project Gradient Noise

3D Graphics

Project Overview

Implemented an Improved 3D Perlin Gradient Noise algorithm, leveraging 12 fixed gradient vectors and quintic interpolation for natural, high-fidelity noise generation. Developed a custom GPU-based pipeline using R32F framebuffers for efficient noise baking and 3D terrain visualization.

Tasks Completed

  • Core Algorithm: Engineered the Improved Perlin Noise logic, utilizing a hardcoded permutation table and fixed gradient vectors.
  • GPU Pipeline: Configured R32F framebuffers to store high-precision noise data, enabling seamless integration into 3D shaders.
  • Procedural Patterns: Developed five distinct noise patterns: Plain, Fractal Sum, Turbulence, Marble, and Wood.
  • Visualization:
    • Designed a 2D texture-mapped quad for noise inspection.
    • Implemented a vertex displacement shader (surface.vert) to dynamically generate 3D terrain from heightmap data.

Reflection

This project deepened my understanding of GPU-based procedural generation and vertex displacement mapping. The primary challenge—remapping [-1, 1] noise values to [0, 1] for texture storage—highlighted the importance of data normalization in graphics pipelines. Optimizing the vertex shader to transform geometry based on real-time texture sampling provided valuable insights into rendering performance and efficient shader design.
