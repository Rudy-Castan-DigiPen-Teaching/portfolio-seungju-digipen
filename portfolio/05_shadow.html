Project Shadow Mapping

3D Graphics

Project Overview

This project focuses on implementing Shadow Mapping, a core technique for real-time 3D rendering. It involves a two-pass rendering pipeline: generating a depth map from the light's perspective and utilizing it within the camera's viewport to calculate accurate shadows.

Tasks Completed

  • Depth Pass: Implemented a WriteDepth shader to store scene depth information in a FrameBuffer from the light's perspective.
  • Shadow Pass: Developed a shadow detection algorithm using ShadowMatrix to transform world coordinates into light space, optimizing visual fidelity.
  • Optimization & Performance: Implemented TexStorage2D for immutable storage, enhancing stability and performance in WebGL environments.
  • Camera System: Refined View and World matrix calculations to ensure precise coordinate transformations and seamless switching between camera modes.

Reflection

Porting the project to a WebGL environment presented unique challenges, particularly regarding GPU memory management and rendering stability. By optimizing texture allocation and transitioning to a more efficient rendering pipeline, I gained a deeper understanding of hardware-level constraints. This project reinforced the importance of mathematical precision in graphics programming and the necessity of tailoring rendering techniques to meet performance targets in web-based applications.
